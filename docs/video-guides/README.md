# OHSMS Video Training Pack

This folder contains the recording-ready material for creating English training videos for the OHSMS application.

## What is included

- `RECORDING_READINESS_CHECKLIST.md`
  Screen recording prerequisites, demo-account needs, and data setup requirements.
- `ENGLISH_MODULE_VIDEO_SCRIPTS.md`
  Recording scripts, screen flow, and narration guide for each module video.

## Important note

These documents are ready for screen recording, but the actual video generation still requires:

- a safe demo login
- seeded demo data for each module
- a screen-recording tool on the machine used for capture

The current machine session does not have a video recorder installed, so this pack is the production-ready handoff for recording the final videos.
