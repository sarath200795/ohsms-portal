# Recording Readiness Checklist

Use this checklist before recording the English module walkthrough videos.

## 1. Recording setup

- Use a stable 16:9 screen size, ideally `1920x1080`
- Record in the Chrome browser unless another browser is required
- Close chat apps, email popups, and personal browser tabs
- Turn off system notifications
- Use a clean desktop background if the full screen will be visible
- Use one consistent browser zoom level, ideally `100%`

## 2. Audio setup

- Record in English
- Use a quiet room and a consistent microphone
- Keep narration slow and instructional
- Avoid improvising feature descriptions that are not visible on screen

## 3. Demo account requirements

- One global admin demo account
- One site manager or user demo account
- One field portal demo account
- One vendor portal demo account

## 4. Demo data required

- At least 2 sites
- 5 to 10 users with mixed roles
- Active PTW records, including pending and closed permits
- Active LOTO procedures and one executable record
- Incident records with CAPA and HIRA links
- Risk assessments with activities, hazards, and revision logs
- Training records with valid, expiring, and expired items
- Consultation records with actions
- Audit plans, findings, and responses
- Improvement proposals in proposed, approved, and completed status
- Inspection templates with assigned schedules and completed history
- Emergency drill records
- Emergency equipment assets with upcoming and overdue inspections
- Contractor companies and workers

## 5. Recording style rules

- Start every video on the module landing page
- Show the route from dashboard or portal before entering the module
- Explain what the module is for in the first 15 seconds
- Move slowly enough for viewers to read labels and values
- Use cursor pauses before major clicks
- Keep each video focused on one module only
- End each video by summarizing the main actions users can perform

## 6. Editing rules

- Add title card with module name
- Add subtitles or on-screen callouts where helpful
- Cut long page loads
- Blur real names, emails, phone numbers, and sensitive IDs if live data is used
- Export each video in `1080p`
